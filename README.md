# Meridian

A precision-instrument clock. Single HTML file, no dependencies.

**Features**
- Live hours, minutes, seconds
- Milliseconds counter running at display refresh rate
- Day, date, and local timezone
- Colons blink in sync with actual second boundaries
- Respects `prefers-reduced-motion`

**Design**
Cool blue-gray palette on a subtle graph-paper grid — visually rooted in navigational charts and scientific timing equipment rather than the usual dark/neon or warm/serif clock aesthetics.

Open `index.html` in any browser.
